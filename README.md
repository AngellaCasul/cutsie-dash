## 📖 About the Game

Cutsie-Dash is an adorable endless runner game where you control a squishy pink creature hopping between lanes to dodge incoming blue cube enemies. The longer you survive, the higher your score — and the faster it gets! Can you beat your personal best?

---

## 🕹️ Controls

| Key | Action |
|-----|--------|
| `↑` / `W` | Move up a lane |
| `↓` / `S` | Move down a lane |
| `Space` | Start the game |
| `P` / `ESC` | Pause / Resume |
